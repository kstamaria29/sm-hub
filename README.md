# sm-hub

Family Hub Expo app now lives at repository root.

## Mobile App

Environment variables:

- `EXPO_PUBLIC_SUPABASE_URL`
- `EXPO_PUBLIC_SUPABASE_ANON_KEY`

Commands:

- `npm install`
- `npm run start`
- `npm run lint`

## Backend (Supabase)

- Use a hosted Supabase project (no Docker required).
- Run SQL files manually in Supabase Dashboard SQL Editor in the exact order listed in `supabase/README.md`.

See `supabase/README.md` for edge function secrets and deployment checklist.
